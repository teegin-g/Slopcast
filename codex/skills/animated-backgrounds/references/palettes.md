## Custom Palette Tips

When matching an existing app's theme:

1. **Extract 2-3 dominant colors** from the user's screenshot or CSS
2. **Darken them 2-3 stops** for the background base (backgrounds should be darker than UI)
3. **Use the brightest color** as the accent/glow source at low opacity (0.05-0.15)
4. **Create atmosphere** by using the accent color in radial gradients behind focal points
5. **Keep contrast low** in the background â€” it should support, not compete with, the UI layer
