  <files>src/styles/glass.css, src/app.css, src/components/ui/GlassPanel.tsx, src/components/ui/GlassCard.tsx, src/components/ui/Vignette.tsx, src/components/ui/GlassPanel.test.tsx, src/components/ui/GlassCard.test.tsx</files>
  <action>
1. Create `src/styles/glass.css` with glassmorphism tokens. Define in `:root` (Slate defaults), then override per `[data-theme]`:

```css
:root {
  /* Sidebar: nearly opaque, Databricks-style solid anchor */
  --glass-sidebar-bg: rgba(var(--bg-deep), 0.92);
  --glass-sidebar-blur: 12px;
  --glass-sidebar-border: rgba(var(--cyan), 0.12);

  /* Content panels: 60-70% opacity — backgrounds are the star */
  --glass-panel-bg: rgba(var(--surface-1), 0.35);
  --glass-panel-blur: 18px;
  --glass-panel-border: rgba(var(--cyan), 0.10);
  --glass-panel-shadow: 0 8px 32px rgba(0, 0, 0, 0.3), 0 0 12px rgba(var(--cyan), 0.05);

  /* Inner cards: subtler layer */
  --glass-card-bg: rgba(var(--surface-2), 0.25);
  --glass-card-blur: 8px;
  --glass-card-border: rgba(var(--border), 0.15);
}
```

Add per-theme overrides for each `[data-theme]` value (synthwave, tropical, moonlight, storm-dusk, hyperborea). Each theme should tint the glass with its accent color (warm pink for Synthwave, green for Tropical, etc.). Check `theme.css` for the exact data-theme attribute values.

Add Mario override: `[data-theme='mario']` sets glass tokens to fully opaque solid surfaces (no blur, no transparency).

2. Update `src/app.css` to import glass.css: add `@import "./styles/glass.css";` after the theme.css import.

3. Create `src/components/ui/GlassPanel.tsx` (COMP-01 — outer card):
- Props: `{ isClassic: boolean; children: React.ReactNode; className?: string; hover?: boolean }`
- If `isClassic`: render a `<div>` with classes `rounded-panel` and `theme-transition` (the existing Mario panel style)
- If NOT isClassic: render glass panel using CSS custom property vars:
  - `background: var(--glass-panel-bg)`
  - `backdrop-filter: blur(var(--glass-panel-blur))`
  - `-webkit-backdrop-filter: blur(var(--glass-panel-blur))`
  - `border: 1px solid var(--glass-panel-border)`
  - `box-shadow: var(--glass-panel-shadow)`
  - Add `rounded-panel` and `theme-transition` classes
  - If `hover` prop is true (default false): add `hover:brightness-110 hover:scale-[1.005] transition-all duration-200`
- Use inline styles for the CSS custom property values (Tailwind arbitrary values may not resolve var() correctly for backdrop-filter). Tailwind classes for everything else.

4. Create `src/components/ui/GlassCard.tsx` (COMP-02 — inner card):
- Props: `{ isClassic: boolean; children: React.ReactNode; className?: string }`
- Same isClassic branching pattern but uses `--glass-card-*` tokens and `rounded-inner` class
- No backdrop-filter on inner cards (performance — per research, limit blur to 3-4 panels)
- Inner cards use `background: var(--glass-card-bg)` and `border: 1px solid var(--glass-card-border)` only

5. Create `src/components/ui/Vignette.tsx`:
- Fixed overlay at z-10: `position: fixed; inset: 0; pointer-events: none; z-index: 10`
- Radial gradient from transparent center to `rgba(0,0,0,0.3)` at edges
- No props needed besides optional `className`
