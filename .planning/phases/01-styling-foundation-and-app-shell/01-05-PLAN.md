| `bg-black/30` | `bg-theme-surface1/40` | Darker overlay |

| `hover:bg-white/15` | `hover:bg-theme-surface2/40` | Hover active |

**CRITICAL RULES:**
1. Inside `isClassic ? '...' : '...'` ternaries, ONLY replace the non-classic (second) branch. Leave classic branches untouched -- they use white/black deliberately for solid retro styling.

3. When you see `text-white` without an opacity modifier, replace with `text-theme-text`.
4. When you see standalone `border-white` (no modifier), replace with `border-theme-border`.
5. Do NOT add new imports -- these are Tailwind classes, not component imports.

Work through each file methodically. Read the file, identify all hardcoded patterns, apply replacements respecting the ternary branches, write the file.
