STYLE-02 — Spacing standardization:
1. Audit all components created in Plans 01/02 for spacing consistency. Enforce compact/dense spacing per user decision:
   - Gaps between sibling panels: `gap-2` (8px) or `gap-3` (12px)
   - Padding inside panels: `p-3` (12px) or `p-4` (16px)
   - Padding inside inner cards: `p-2` (8px) or `p-3` (12px)
   - Sidebar item padding: `px-3 py-1.5` (compact)
   - Section spacing: `space-y-1` within nav, `space-y-2` between sections
2. Add spacing documentation as CSS comments in `src/app.css`:
```css
/* Spacing scale (compact density):
   4px  — micro gap (icon-to-text)
   8px  — inner gap (within cards, between list items)
   12px — standard gap (between cards, panel padding)
   16px — section gap (between major sections)
   24px — area gap (between content regions)
   32px — page margin (content area padding)
   48px — hero spacing (landing page only)
*/
```

STYLE-03 — Typography hierarchy:
3. Add typography utility layer in `src/app.css` using Tailwind @layer:
```css
@layer components {
  .typo-h1 { @apply text-2xl font-light tracking-tight text-theme-text; }
  .typo-h2 { @apply text-xl font-light tracking-tight text-theme-text; }
  .typo-h3 { @apply text-base font-normal text-theme-text; }
  .typo-body { @apply text-sm font-normal text-theme-text; }
  .typo-caption { @apply text-xs text-theme-muted; }
  .typo-label { @apply text-xs font-medium uppercase tracking-wider text-theme-muted; }
  .typo-value { @apply font-mono tabular-nums; }
}
```
- Headers: font-weight 300-400, larger sizes, tracking-tight — "elegant, lets glass and colors do the talking"
- Labels: small, muted, uppercase tracking — finance app convention
- `.typo-value`: monospace + tabular numerals for financial alignment (use `font-mono tabular-nums`)
- Font family: Inter is already loaded via Google Fonts, use it as primary via system stack. Set in @theme: `--font-sans: 'Inter', ui-sans-serif, system-ui, sans-serif`

4. Apply typography classes to sidebar components:
   - Section names: `typo-label` when collapsed (hidden), `typo-body font-medium` when expanded
   - Group names: `typo-body`
   - Well count badges: `typo-caption`
