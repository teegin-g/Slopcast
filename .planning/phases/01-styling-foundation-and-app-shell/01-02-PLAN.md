1. Create `src/hooks/useSidebarNav.ts`:
```typescript
type Section = 'wells' | 'economics' | 'scenarios';

export function useSidebarNav() {
  const [searchParams, setSearchParams] = useSearchParams();
  const section = (searchParams.get('section') as Section) || 'wells';
  const setSection = (s: Section) => {
    setSearchParams(prev => { prev.set('section', s); return prev; }, { replace: true });
  };
  return { section, setSection };
}
```
This is the adapter layer. It bridges URL search params to the sidebar UI. The AppShell will then map `section` to `useSlopcastWorkspace`'s `designWorkspace` and `viewMode` state.

Mapping: `wells` -> designWorkspace='WELLS', `economics` -> designWorkspace='ECONOMICS', `scenarios` -> viewMode='ANALYSIS'

```
  {viewport === 'mobile' && (
    <MobileDrawer open={mobileDrawerOpen} onClose={() => setMobileDrawerOpen(false)}>
      <Sidebar ... collapsed={false} />
    </MobileDrawer>
  )}

  {/* Mobile header bar with hamburger menu (mobile only) */}
  {viewport === 'mobile' && (
    <div className="fixed top-0 left-0 right-0 z-30 h-12 flex items-center px-3" style={{background: 'var(--glass-sidebar-bg)'}}>
      <button onClick={() => setMobileDrawerOpen(true)}>hamburger icon</button>
      <span>Section name</span>
    </div>
  )}

  {/* Content area */}
  <main className="relative z-20 flex-1 overflow-y-auto">
    <div className="p-3 max-w-[1920px] mx-auto w-full">
      {/* Content rendered based on section */}
    </div>
  </main>
</div>
```

Key logic:
- Use `useViewportLayout()` for responsive behavior
- Use `useSidebarNav()` for section state
- Map section to useSlopcastWorkspace state: sync `section` with `designWorkspace` and `viewMode`

  - On mount: read from URL params and set initial workspace state
  - Avoid infinite loops: compare before setting, use `replace: true`
- Auto-collapse sidebar on 'mid' viewport (icon-only mode)
- Desktop: manual toggle persists to localStorage
- Sidebar collapse state: `const [collapsed, setCollapsed] = useState(viewport === 'mid')`
